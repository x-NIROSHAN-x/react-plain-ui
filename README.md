# @x-niroshan-x/react-plain-ui

> A bunch of react ui components with plain ui design

[![NPM](https://img.shields.io/npm/v/@x-niroshan-x/react-plain-ui.svg)](https://www.npmjs.com/package/@x-niroshan-x/react-plain-ui) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save @x-niroshan-x/react-plain-ui
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from '@x-niroshan-x/react-plain-ui'
import '@x-niroshan-x/react-plain-ui/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
